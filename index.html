# Our JavaScript redirection code

This feature is for you to connect other apps to Redirection. For example, you can connect host.vercel.app to store documents!
https://replace.vercel.app/gotohost.vercel.app It will redirect you to: https://host.vercel.app/yourdocument Note: This is just an example.
